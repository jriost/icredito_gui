# **App Name**: ReactFlow

## Core Features:

- Login: User authentication via Django API.
- Dashboard: A responsive dashboard with a collapsible sidebar navigation.
- Data Tables: Display data fetched from Django API in responsive tables.
- Form Handling: Submit forms to the Django API and handle responses.
- Dark/Light Mode: Toggle between dark and light themes for the entire application.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a modern and sophisticated feel.
- Background color: Light gray (#F5F5F5) for light mode and dark gray (#303030) for dark mode, both subtly desaturated from the primary hue.
- Accent color: Teal (#009688) to provide a contrast against the indigo, creating highlights for interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral look that is easy to read in both light and dark modes.
- Use minimalist icons from a library like Material Icons, ensuring they are clear and understandable. Use filled icons for the active state in the navigation.
- Implement a responsive grid layout with breakpoints for various screen sizes. The sidebar should collapse into a hamburger menu on smaller screens.
- Use subtle animations and transitions for UI elements such as menu toggles and content loading to enhance user experience without being distracting.
